# 音乐文件说明

请将你的圣诞主题音乐文件放在此目录下，并命名为 `christmas-music.mp3`。

## 推荐的圣诞音乐

- Jingle Bells
- Silent Night
- We Wish You a Merry Christmas
- Deck the Halls
- O Holy Night

## 格式要求

- 格式: MP3
- 比特率: 128-256 kbps (推荐)
- 长度: 建议3-5分钟，循环播放效果最佳

## 替换音乐

1. 将你的音乐文件重命名为 `christmas-music.mp3`
2. 替换此目录中的现有文件
3. 重新启动开发服务器
